# PowerScrolls

[Spigot Page](https://www.spigotmc.org/resources/%E2%AD%90-powerscrolls-itemscore-addon.130453/)

This plugin lets you create power scrolls and chain them with items to upgrade and add abilities to them!

plugin was mainly made to be an open source addon example for [ItemScore](https://www.spigotmc.org/resources/%E2%9A%94%EF%B8%8F%E2%AD%95itemscore-premium%E2%AD%90abilities%E2%9C%85events%E2%9C%85stats%E2%9C%85create-items-with-a-prompt-1-8-1-21-8.117905/)!
